# Cyclistic Bike Share Analysis - Google Data Analysis Capstone Project

This project is the final capstone of the Google Data Analysis Class from 
coursera. The goal of the project is to do an analysis to generate business 
recommendations for Cyclistic based on the different behaviors of its riders 
with the ultimate marketing strategy to convert casual riders into annual 
memberships. 

## Key Insights 
* All riders peak during the summer months
* Casual riders take longer rides and more rides on the weekend
* During the day, Casual riders have a single peak around commuting hours

## Tools Used
